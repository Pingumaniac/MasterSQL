How could data from a table be deleted in SQL?

To delete data in the table in SQL, we need to write a query in the following format:

DELETE FROM (Write the table name here) WHERE condition;

Here note that the WHERE clause is needed unless you want to delete all the rows in the table.

For example, assume a table named "Current_iPads," which contains data about the iPads in which Apple is currently selling.

When executing the following query:

SELECT * FROM Current_iPads ORDER BY Display_size DESC;

The following table will be popped up.

# Model_name Manufacturer Display_size
1 iPad Pro 12.9 (5th generation) Apple 12.9
2 iPad Pro 11 (3th generation) Apple 11
3 iPad Air (4th generation) Apple 10.9
4 iPad (9th generation) Apple 10.2
5 iPad mini (6th generation) Apple 8.3
6 iPad mini (5th generation) Apple 7.9
Note that the iPad mini (5th generation) is currently discontinued. We can execute the following query to update the table:

DELETE FROM Current_iPads WHERE Model_name = 'iPad mini (5th generation)';

When executing the selection query again, the new table will be:

# Model_name Manufacturer Display_size
1 iPad Pro 12.9 (5th generation) Apple 12.9
2 iPad Pro 11 (3th generation) Apple 11
3 iPad Air (4th generation) Apple 10.9
4 iPad (9th generation) Apple 10.2
5 iPad mini (6th generation) Apple 8.3
