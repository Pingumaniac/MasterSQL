How could tables be dropped in SQL?

Dropping a table from a database in SQL is much simpler than creating a table in the database in SQL.

You need to write a query in the following format:

DROP TABLE (Write the table name here);

An example query to drop a table named 'Duty' would be:

DROP TABLE Duty;
