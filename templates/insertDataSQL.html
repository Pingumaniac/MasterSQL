How could we insert data into a table in SQL

There are two ways to insert data into a table.

One way is to specify all the names of the attributes for the values you are going to insert into the table.

In this case, you can insert data into the table in the following format:

INSERT INTO (Write the table name here) (name_of_first_attribute, name_of_second_attribute, name_of_third_attribute, ...) VALUES (value_for_first_attribute, value_for_second_attribute, value_for_third_attribute, ...);

If you are going to insert values for all the attributes, then you can instead insert data into the table in this format:

INSERT INTO (Write the table name here) VALUES (value_for_first_attribute, value_for_second_attribute, value_for_third_attribute, ..., value_for_last_attribute);

A case in point, assume that there is a table named 'SmartPhones' with three attributes: Model_name, Manufacturer, Display_size.

When executing the following query:

SELECT * FROM SmartPhones ORDER BY Display_size DESC;

The following table will be popped up.

# Model_name Manufacturer Display_size
1 iPhone 13 Pro Max Apple 6.7
2 iPhone 13 Pro Apple 6.1
3 iPhone 13 Apple 6.1
4 iPhone 13 Mini Apple 5.4
5 iPhone 12 Mini Apple 5.4
6 iPhone 11 Mini Apple 5.4

If you want to insert a row in the table about iPhone SE2 which ismanufactured by Apple and has a display size of 4.7 (inches), then you can write the insert query in either two ways:

  1. INSERT INTO SmartPhones (Model_name, Manufacturer, Display_size) VALUES ('iPhone SE2', 'Apple', 4.7);
  2. INSERT INTO SmartPhones VALUES ('iPhone SE2', 'Apple', 4.7);

If you execute the selection query again, the table will contain another row with data about iPhone SE2.

# Model_name Manufacturer Display_size
1 iPhone 13 Pro Max Apple 6.7
2 iPhone 13 Pro Apple 6.1
3 iPhone 13 Apple 6.1
4 iPhone 13 Mini Apple 5.4
5 iPhone 12 Mini Apple 5.4
6 iPhone 11 Mini Apple 5.4
7 iPhone SE2 Apple 4.7
