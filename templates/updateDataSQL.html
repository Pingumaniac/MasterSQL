How could data in the table be updated in SQL?

To update data in the table in SQL, we need to write a query in the following format:

UPDATE (Write the table name here) SET attribute1 = value1, attribute2 = value2 WHERE condition;

Here note that the WHERE clause is needed unless you want to update the attribute value for all the rows in the table.

For example, assume that there is a table named "iPads," which contains data about the iPads Apple is currently selling.

When executing the following query:

SELECT * FROM iPads ORDER BY Display_size DESC;

The following table will be popped up.

# Model_name Manufacturer Display_size
1 iPad Pro 12.9 Apple 12.9
2 iPad Pro 11 Apple 11
3 iPad Air Apple 10.9
4 iPad Apple 10.2
5 iPad mini Apple 7.9
Note that we shall update the display size for the iPad mini from 7.9 to 8.3. We can execute the following query to update the table:

UPDATE iPads SET Display_size = 8.3 WHERE Model_name = 'iPad mini';

When executing the selection query again, the new table will be:

# Model_name Manufacturer Display_size
1 iPad Pro 12.9 Apple 12.9
2 iPad Pro 11 Apple 11
3 iPad Air Apple 10.9
4 iPad Apple 10.2
5 iPad mini Apple 8.3
